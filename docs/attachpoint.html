Attach Point Tool
by Matharoo

Overview



Attach Point Tool helps you set up attach points on sprite sheets. It exports code that you can put in your project and retrieve in-game.



Workflow

  1. Import sprite sheet(s)
  2. Set up attach points and their angles
  3. Export code


Exported Code

The exported code will export a nested array structure, in this format:



Sprite : [ Frame0 : [x, y, angle], Frame1 : [x, y, angle], Frame3 : [x, y, angle], // More frames... ] // More sprites (if more than one imported)

The coordinates are relative to the top-left corner of each frame.
